### 二、详细每日行程

**Day 1 — 10月1日（周四）：抵达洛杉矶**
- 搭乘国际航班抵达洛杉矶国际机场（LAX），预计傍晚6点左右入境。
- 乘坐机场穿梭巴士前往酒店办理入住，就近在酒店附近餐厅用简餐，之后休息调整时差。
- 交通：机场穿梭巴士（约$10-15 / CNY 70-110）
- 住宿：Hampton Inn Los Angeles Airport（靠近LAX，便于抵达后快速入住休息）


**Day 2 — 10月2日（周五）：洛杉矶环球影城全天**
- 上午：提前在网上购买好莱坞环球影城单日票（约$109-154 / CNY 780-1,110），优先体验“哈利波特魔法世界”园区，朋友特别推荐过这里的黄油啤酒和禁忌之旅。
- 下午：继续游玩“超级任天堂世界”和影城之旅（Studio Tour），提前预留快速通行券以节省排队时间。
- 交通：酒店打车往返（约$30-40 / CNY 220-290）


**Day 3 — 10月3日（周六）：洛杉矶市区经典游**
- 上午：参观好莱坞星光大道（免费），寻找最喜欢的影星名字；之后前往格里菲斯天文台，俯瞰洛杉矶全景和好莱坞标志（天文台入场免费，天文馆表演需购票约$7-10 / CNY 50-70）。
- 下午：前往圣莫尼卡海滩，在码头上散步、看太平洋日落，附近有餐厅可用晚餐。
- 交通：租车自驾或Uber出行


**Day 4 — 10月4日（周日）：洛杉矶 → 拉斯维加斯**
- 上午：在洛杉矶退房后，租车自驾前往拉斯维加斯（全程约4小时车程，约270英里 / 435公里）。
- 下午：抵达后入住酒店，稍作休息。
- 晚上：步行游览拉斯维加斯大道，观看百乐宫音乐喷泉（免费，每15-30分钟一场）；顺路打卡“欢迎来到拉斯维加斯”标志牌（免费）。
- 交通：租车自驾（约4小时，加油约$30-40 / CNY 220-290）
- 住宿：The STRAT Hotel, Casino & Tower（位于大道北端，性价比高）


**Day 5 — 10月5日（周一）：大峡谷国家公园南缘一日游**
- 全天：参加当地旅行团的大峡谷南缘巴士一日游（拉斯维加斯往返，约$100-150 / CNY 720-1,080），含导游和午餐。
- 约早上6点从酒店接车出发，在大峡谷南缘停留约3小时，可欣赏Mather Point等观景台，下午约6-7点返回拉斯维加斯。
- 交通：跟团大巴（含接送）


**Day 6 — 10月6日（周二）：拉斯维加斯 → 旧金山**
- 上午：退房后前往拉斯维加斯机场（LAS），搭乘早班航班飞往旧金山（SFO），航程约1.5小时。
- 下午：抵达后乘机场快线至市中心，入住酒店后步行游览渔人码头（免费），品尝Boudin面包店的酸面包海鲜浓汤（约$15-20 / CNY 110-145）；随后步行至九曲花街（免费）拍照打卡。
- 交通：航班（约$80-150 / CNY 580-1,080）+ 机场快线（约$10 / CNY 70）
- 住宿：Holiday Inn San Francisco — Golden Gateway（靠近渔人码头，步行可达）


**Day 7 — 10月7日（周三）：旧金山经典一日**
- 上午：乘公交车或打车前往金门大桥（免费）观景拍照，可在Battery Spencer观景点欣赏大桥全貌。
- 下午：乘渡轮前往恶魔岛参观（需提前网上预订门票，约$45 / CNY 325），体验监狱历史导览。
- 晚上：在联合广场周边逛街和用晚餐，体验旧金山铛铛车（约$8 / CNY 60）。
- 交通：公交/渡轮/铛铛车 + Uber（合计约$50-70 / CNY 360-500）


**Day 8 — 10月8日（周四）：旧金山深度体验**
- 上午：参观加州科学博物馆（California Academy of Sciences），位于金门公园内，门票约$40-50 / CNY 290-360。
- 下午：漫步金门公园，参观日本茶园（Japanese Tea Garden，门票约$15-20 / CNY 110-145）。
- 晚上：乘Uber前往双子峰俯瞰旧金山城市夜景（免费）。
- 交通：公交 + Uber（约$20-30 / CNY 145-220）


**Day 9 — 10月9日（周五）：旧金山自由活动 / 购物**
- 上午：可前往旧金山奥特莱斯购物（如Livermore Premium Outlets），乘BART+接驳车可达（往返约$15-20 / CNY 110-145）。
- 下午：自由安排，可重游喜欢的景点或逛旧金山唐人街（免费）。
- 晚上：整理行李，为次日返程做准备。
- 交通：BART + 接驳车 / Uber


**Day 10 — 10月10日（周六）：旧金山 → 返程**
- 上午：酒店退房后，乘BART（湾区捷运）前往旧金山国际机场（SFO）（约$10 / CNY 70）。
- 搭乘国际航班返回中国，行程结束。
- 交通：BART至SFO
