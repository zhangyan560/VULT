## 硬件与系统要求

- **必须有 Mac**：iOS 开发只能在 macOS 上进行，无法在 Windows/Linux 原生运行
- 推荐配置：M1 芯片以上 MacBook/iMac，8GB RAM 起步（开发大型项目建议 16GB+）
- Xcode 占用空间大（15GB+），需要预留充足磁盘空间

## Xcode 安装

1. 从 Mac App Store 免费下载 Xcode（当前版本支持 iOS 18）
2. 安装时间较长，期间会有"卡住"现象，属正常
3. 安装完成后会附带 Swift、SwiftUI 框架，无需单独安装
4. Xcode 集成了：代码编辑器、调试器、模拟器、性能分析工具（Instruments）

## Apple Developer Program

| 类型 | 费用 | 限制 |
|------|------|------|
| 免费账号 | 0 | 可在真机测试（有效期 7-14 天），不可上架 |
| 个人开发者 | $99/年 | 可上架 App Store，最多注册为个人 |
| 组织开发者 | $299/年 | 可用公司名称上架，支持多人协作 |

> [!important] 仅学习/测试不需要付费账号，只有上架 App Store 才需要 $99/年。

## 证书与签名配置

```
证书配置流程：
developer.apple.com → Certificates, Identifiers & Profiles
    ↓
1. Certificates（上传本地生成的 CSR 文件）
2. Identifiers（创建唯一 Bundle ID，格式：com.公司名.App名）
3. Provisioning Profiles（关联证书 + Bundle ID + 设备）
```

> [!tip] 现代 Xcode 版本支持"Automatically manage signing"，可自动处理代码签名，新手无需手动配置。

## Swift/SwiftUI/UIKit 学习路径

```
学习时间轴：
Week 1-2: Swift 基础
    变量(var/let)、数据类型、函数、控制流、枚举

Week 3-4: 面向对象与协议
    Class vs Struct（引用类型 vs 值类型）、POP 协议导向编程

Month 2: SwiftUI 核心
    @State/@Binding 状态管理、View 组合、@Observable 宏、NavigationStack

Month 3: 进阶功能
    网络请求（URLSession + async/await）
    数据持久化（SwiftData / UserDefaults）
    推送通知（APNs）
    支付集成（StoreKit）

Month 4+: 独立开发能力
    整合所有模块 → 构建完整 MVP → TestFlight 测试 → 上架
```

## 关键辅助工具

- **SF Symbols**：苹果官方提供数千个原生风格图标，可直接在 App 中引用
- **模拟器（Simulator）**：可测试不同屏幕尺寸、深色模式、地理位置等场景
- **Instruments**：性能分析工具，用于检测内存泄漏、CPU 使用率
- **TestFlight**：Beta 测试分发平台，内部100人/外部10,000人
