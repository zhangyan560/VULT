## 付费广告渠道对比

| 平台 | 核心优势 | 适用阶段 | 成本参考 | 特别说明 |
| :--- | :--- | :--- | :--- | :--- |
| **TikTok Ads** | 短视频形式天然契合演示，新账号更易自然爆火 | MVP 验证 → 增长期 | 冷启动 $100-200 测试 | 视频内容质量决定一切 |
| **Google UAC** | 覆盖全 Google 生态，算法读取 App 元数据 | 增长期 → 成熟期 | 按安装付费，CPI 差异大 | ASO 质量直接影响广告效率 |
| **Meta Ads** | 庞大用户基数，定位系统成熟 | 增长期 | 受隐私政策影响，iOS 效果下降 | 更适合安卓 + 内容类 App |

> [!note] 核心原则
> 广告是验证转化路径的工具，不是启动增长的唯一手段。先通过有机渠道验证产品，再用付费广告放大效果。

## 网红营销策略

### ROI 案例（真实数据）

> Social Wizard AI App：支付微型主播 **$120** → 视频获得 **200 万次播放** → 带来**数万美元**收入

### 操作方法

1. **寻找利基市场的微型博主**（Niche Micro-influencer）
   - 不追顶级网红，找与目标用户高度重叠的中小型账号
   - 粉丝量 5k-100k 的账号互动率往往更高

2. **制定"暂停"策略**
   - 视频前 3 秒必须够抓人（漂亮视觉、强对比、悬念感）
   - 让用户在刷屏时停下来

3. **演示而非推销 (Show don't tell)**
   - 展示 App 如何解决真实问题（如：真实演示用 App 回复私信的流程）
   - 勿直接说"下载我的 App"

## 内容营销与有机增长

### TikTok / Instagram 有机流量策略

```
核心公式：高频产出 × 正确格式 = 确定性爆发

目标：每天发布 3+ 短视频
逻辑：如果一周发布 700 个视频，其中一个爆火不再是运气
```

**内容选题方向**：
- 演示型：展示 App 解决具体问题的过程
- 数据震撼型：$120 换来 200 万播放？我用这款 App 做到了
- 教程型：如何用 [工具类 App] 做到 XX
- 对比型：有/没有这个 App 的区别

### Reddit 有机获客

1. 监控与你产品相关的关键词讨论
2. 找正在讨论该痛点的帖子
3. 在回复中诚恳推荐 App 作为解决方案
4. Reddit 内容易被 Google 收录 → 长期 SEO 价值

## 零预算冷启动 SOP

```
Step 1：构建 MVP → 解决自己的实际问题
Step 2：ASO 优化 → 利用商店自然搜索（65% 安装来自搜索）
Step 3：手动获客 → Reddit / Twitter 潜水，定向推荐
Step 4：高频内容 → TikTok/Instagram 每天 3+ 视频
Step 5：免费试用 → 获取真实用户行为数据和评论
Step 6：引导评价 → 建立社会证明，触发更多下载
```

## AdMob 广告集成与优化

### 集成策略

- 选择能提供最高 eCPM 的广告网络（AdMob 为首选）
- 根据 App 类型选择广告位：
  - 激励视频 > 插屏 > 横幅（eCPM 从高到低）

### 优化要点

| 优化项 | 具体操作 |
| :--- | :--- |
| 混合变现 | 对高价值用户推销订阅，对低付费用户展示激励视频 |
| 素材更新 | 定期更换广告素材防止"广告疲劳" |
| 频率控制 | A/B 测试找到广告频率与留存率的最优平衡 |
| 数据监控 | Mixpanel / PostHog 追踪广告渠道盈利能力 |

> [!warning] 平衡用户体验
> 广告过载会直接拉高卸载率。宁可广告少一点，留存高一点。
