# 应用市场评论爬取与分析

> Google Play 与 Apple App Store 应用评论、评分、版本迭代反馈的采集与分析方案。

## 一、为什么采集应用市场评论

- **用户需求挖掘**：识别用户最痛的功能缺失点
- **竞品差异分析**：对比竞品 App 的差评集中领域
- **版本迭代追踪**：监控评分随版本更新的变化曲线
- **情感倾向分析**：NLP 提炼正/负面评价的关键词

---

## 二、Google Play 采集方案

### 2.1 google-play-scraper（推荐）

```python
from google_play_scraper import app, reviews, Sort

# 获取 App 基本信息
result = app(
    'com.spotify.music',
    lang='en',
    country='us'
)
print(result['title'], result['score'], result['installs'])

# 批量获取评论
result, continuation_token = reviews(
    'com.spotify.music',
    lang='en',
    country='us',
    sort=Sort.NEWEST,
    count=5000,
    filter_score_with=None  # None = 所有评分
)
# result 是包含 userName, score, content, at(时间) 的列表
```

### 2.2 SerpApi（结构化 API）

- 提供 Google Play Product API
- 支持按 App ID + 国家/语言过滤
- 按次计费，适合生产环境

### 2.3 Octoparse（无代码）

- 提供 Google Play 专用任务模板
- 自动模拟滚动 + 点击"显示更多"
- 可采集数千条评论，导出 CSV/Excel

## 三、Apple App Store 采集方案

### 3.1 app-store-scraper

```python
from app_store_scraper import AppStore

app = AppStore(country='us', app_name='spotify', app_id='324684580')
app.review(how_many=1000)

import pandas as pd
df = pd.DataFrame(app.reviews)
# 包含 userName, rating, title, review, date 字段
```

### 3.2 iTunes RSS Feed（官方接口）

Apple 提供公开 RSS 接口，获取最新 500 条评论：
```
https://itunes.apple.com/{country}/rss/customerreviews/id={app_id}/sortBy=mostRecent/json
```

## 四、双平台数据整合

```python
import pandas as pd

# Google Play 评论
gp_reviews = pd.DataFrame(gp_result)[['userName', 'score', 'content', 'at']]
gp_reviews['platform'] = 'Google Play'

# App Store 评论
as_reviews = pd.DataFrame(app.reviews)[['userName', 'rating', 'review', 'date']]
as_reviews.rename(columns={'rating': 'score', 'review': 'content', 'date': 'at'}, inplace=True)
as_reviews['platform'] = 'App Store'

# 合并
all_reviews = pd.concat([gp_reviews, as_reviews], ignore_index=True)
```

## 五、评论数据分析流程

### 5.1 数据清洗

```python
# 去除空评论
df = df[df['content'].str.len() > 10]
# 去重
df.drop_duplicates(subset=['content'], inplace=True)
# 时间转换
df['at'] = pd.to_datetime(df['at'])
```

### 5.2 情感分析

```python
from transformers import pipeline

# 使用预训练模型
sentiment = pipeline("sentiment-analysis", model="distilbert-base-uncased-finetuned-sst-2-english")

df['sentiment'] = df['content'].apply(
    lambda x: sentiment(x[:512])[0]['label']
)
```

### 5.3 关键词提取 + 词云

```python
from wordcloud import WordCloud
import matplotlib.pyplot as plt

negative_reviews = ' '.join(df[df['score'] <= 2]['content'])
wc = WordCloud(width=800, height=400).generate(negative_reviews)
plt.imshow(wc)
plt.axis('off')
plt.title('差评关键词词云')
plt.show()
```

## 六、版本迭代监控方案

| 步骤 | 操作 |
|------|------|
| 1 | 定时任务（每日/每周）触发采集脚本 |
| 2 | 采集新增评论，带版本号字段 |
| 3 | 计算各版本平均评分、差评率 |
| 4 | 版本发布后 7 天内评分骤降 → 触发预警 |
| 5 | 聚类差评关键词 → 识别 Bug 类型 |
