# 落地页与跟踪技术

![[落地页与跟踪技术-infographic.png|800]]

## 高转化落地页结构

### 标准结构（套利专用）

```
1. 强力标题（6-8 个单词）
   → 大胆直接，清晰陈述价值主张

2. 视觉焦点
   → 高质量头像 / 产品图 / 短视频

3. 利益点展示（Benefits, NOT Features）
   → "平均节省三小时" vs "具有计时功能"

4. 内容主体（≤500 字）
   → 病毒式脚本 / 故事叙事 / 奖学金信息

5. 社会证明
   → 评论、证言、用户数量

6. 单一 CTA（行动呼吁）
   → 清晰的下一步按钮

7. 广告位（AdSense/AdX）
   → 嵌入相关搜索广告（RSOC）
```

> [!tip] 内容长度
> 套利场景控制在 **500 字以内**。太长会降低点击率；太短缺乏价值主张。

### 图库模式（Gallery Mode）——利润放大器

```
用户进入图库落地页（7-12 张图片）
        ↓
每次点击"下一页" = 1 次新的页面加载
        ↓
Google AdSense = 1 次新的广告展示
        ↓
1 个用户 × 11 次翻页 = 11 次广告收入
```

- 图片选题：好奇心驱动（"第 X 张让你惊讶"）
- 每页少量文字 + 广告位 + 下一页按钮
- 用户停留时间长 → Google 认为内容有价值

## AdSense / AdX 广告位布局

### 布局原则

| 方式 | 说明 | 效果 |
|------|------|------|
| 原生集成 | 广告看起来像内容（RSOC 嵌入博客文章中） | 高 CTR |
| 多页面展示 | 图库模式，用户翻页时展示 | 高总展示数 |
| Feed 流布局 | 内容下方类 Taboola 布局，好奇心引导 | 中 CTR，自然 |

### AdX vs AdSense 选择

| 维度 | AdSense | AdX（Google Ad Manager） |
|------|---------|------------------------|
| 门槛 | 低，容易申请 | 高，需通过合作伙伴 |
| 稳定性 | 容易触发"广告限制" | 对流量波动容忍度高 |
| 适合规模 | 小流量测试 | 大流量规模化运营 |
| 封号风险 | 较高 | 较低 |

> [!important] 建议
> 测试阶段用 AdSense，规模化后切换到 AdX。

## 核心指标优化

### CTR 优化

1. **有机流量平衡**：引入 SEO 流量稀释纯付费流量，避免 CTR 异常（纯付费 CTR 20-27% 会被怀疑）
2. **素材迭代**：参考 Meta Ad Library，用 Canva 微调素材，降低买入 CPC
3. **A/B 测试**：标题、CTA 按钮颜色、图片位置

### RPM / CPC 优化

- 通过 ClickFair 报告找出高价值关键词
- 向 Feed 供应商咨询 Untapped Verticals（竞争小、支付高）
- 地理定向：优先美国、英国、加拿大、澳大利亚

### 用户行为深度优化

| 策略 | 实现方式 | 效果 |
|------|---------|------|
| 增加停留时间 | 病毒式脚本、故事叙事、奖学金内容 | Google 认为内容真实有价值 |
| 流量复访 | 推送通知插件（"流量银行"） | 老用户回访 = 极高质量信号 |
| 减少跳出率 | 图库模式、内容分页 | 增加每用户 PV 和广告展示 |

## 追踪工具体系

```
追踪层次：
├── Google Analytics（官方工具，监控流量来源和用户质量）
├── ClickFair（关键词报告 + 回传功能，核心优化工具）
├── System1 / Tonic 后台（Feed 供应商报告，查看关键词支付）
├── Meta Ad Library（竞品素材研究）
└── 推送通知插件（流量银行管理）
```

### 利润实时计算

```python
# 每日利润监控
收入 = Google AdSense/AdX 后台数据
支出 = 广告平台（Meta/Taboola）花费
IVT 扣款 = 通常 3-5%（正常范围）

实际利润 = 收入 × (1 - IVT率) - 支出
```

## 推送通知策略（流量银行）

> [!tip] 流量银行
> 付费买来的流量是一次性的，但推送通知订阅者是永久资产。

1. 在落地页安装推送订阅弹窗
2. 付费流量进站 → 部分用户订阅推送
3. 停止付费后 → 通过推送免费唤回用户
4. 回头客流量质量高 → Google 信任度高 → 更低 IVT 扣款
