# 套利模式解析

## 底层商业逻辑

**搜索套利**的核心是"流量转售"：以低价买入流量，以高价卖出广告点击，赚取价差。

### 运作流程

```
1. 买入流量
   Meta / TikTok / Taboola / Outbrain / LinkedIn
   → 低成本点击（如 $0.10/click）

2. 引导路径
   中间页 / 落地页
   → 含 RSOC（Related Search on Content）或图库

3. 卖出流量（变现）
   用户点击 Google AdSense / AdX 搜索广告
   → Google 支付 CPC（如 $1.00/click）

利润 = 卖出 CPC - 买入 CPC（× 规模）
```

### 利润来源

| 来源 | 说明 |
|------|------|
| 地理位置溢价 | 同一用户在美国比在非洲价值高 10 倍+ |
| 垂直领域溢价 | 牙科/金融/保险广告主出价高 |
| 流量质量溢价 | 高质量来源（LinkedIn）比低质廉价流量值钱 |
| 多页浏览放大 | 图库模式：1 个用户 × 11 次点击 = 11 次展示 |

## 收入公式与关键指标

### 核心利润公式

```
利润 = 总收入（AdSense/AdX）- 总支出（广告费 + 基础建设）
```

### 关键指标体系

| 指标 | 全称 | 含义 | 套利中的意义 |
|------|------|------|------------|
| ROAS | Return on Ad Spend | 广告支出回报率 | 衡量整体盈利效率 |
| 买入 CPC | Cost Per Click（买） | 付给流量平台的费用 | 成本端，要压低 |
| 卖出 CPC | Cost Per Click（卖） | Google 支付给你的费用 | 收入端，要提高 |
| RPM | Revenue Per Mille | 每千次展示收入 | 页面变现效率 |
| CTR | Click-Through Rate | 点击率 | 套利页 CTR 通常 20-27% |
| IVT | Invalid Traffic | 无效流量扣款 | 正常损耗，$12K 收入扣 $500 属于可接受范围 |

## 2024-2025 年 Google 政策风险

> [!warning] 政策收紧趋势
> Google 到 2024 年已能追踪用户访问前后完整路径，"游戏化系统"行为将触发封号。

### 政策新变化

1. **用户路径监控加强**：Google 追踪用户进站前后的完整路径。无实际阅读行为、仅为点击广告的流量被判定为"Gaming the system"→ 封号。

2. **相关搜索（RSOC）审核趋严**：允许内容页展示搜索广告，但机器人点击增多，Google 对此的审核更严格。

3. **"Loading"打击升级**：使用代理服务器（Proxy）+ 指纹浏览器进行虚假点击 → 即时封号。

### 核心风险

| 风险 | 触发条件 | 后果 |
|------|---------|------|
| 账号封禁 | 纯付费流量、无有机流量、虚假点击 | AdSense/AdX 账号永久封禁 |
| 资金冻结 | 收入被判定为无效 | Google 扣除已结算收入 |
| 利润压缩 | 流量竞争加剧，买入 CPC 上涨 | 套利空间缩小甚至亏损 |
| 广告限制 | 转化质量不达标 | 账号被限制展示广告 |

## 现代套利成功模型

> [!tip] 2025 年可持续套利公式
> **SEO 有机流量（基础）+ 高质量社交/原生广告（增量）+ 多页图库（利润放大）**

- 不再是简单低买高卖
- 需要精细化关键词优化
- 合规创意素材（不使用诱导性标题）
- 有机流量支撑（4-5 个月 SEO 积累后才开始跑付费）
