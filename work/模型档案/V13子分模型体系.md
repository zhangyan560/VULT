# V13子分模型体系

> [!info]
> V13新增130个子分模型，经筛选后78个入模，使用率60%。子分与主模型融合是V13的核心设计思路，捕捉主模型遗漏的特定维度风险。

## 子分体系总览

| 指标 | 数值 |
|------|------|
| 新增子分总数 | 130个 |
| 最终入模数 | 78个 |
| 使用率 | 60% |
| 增益最大单子分 | 流入系列 9m1d，+4.7pp |

## 重要子分专项说明

### 增益最大：流入系列（9m1d +4.7pp）

> [!important]
> 流入系列子分是V13增益最大的单一子分类型。9m1d（9期MOB1Day1）子分增益+4.7pp，捕捉用户在不同MOB期入逾当天的风险特征，精准识别早期违约信号。

流入系列4个模型：
- MOB1D1：MOB1第1天入逾预测
- MOB3D1：MOB3第1天入逾预测
- MOB6D1：MOB6第1天入逾预测
- MOB9D1：MOB9第1天入逾预测

### 弹性模型（+2.7pp）

> [!important]
> 弹性模型（MOB1to3）增益+2.7pp。核心思路：用MOB1已逾期用户的后续MOB3行为建模，预测"短期逾期是否会演变为中期逾期"，捕捉弹性还款用户。

### 金额加权子分（1000w +2.6pp）

金额加权子分对高金额样本赋予更大权重，优化高额度客群的风险排序：
- 1500w权重版：适用超高额度客群
- 1000w权重版：增益+2.6pp，综合效果最优
- 500w权重版：适用中等额度客群

### 反欺诈子分（FPD31 +1.65pp）

> [!important]
> FPD31反欺诈子分增益+1.65pp，在二次风控FPD31模型中重要性高达38.9%，是二次风控最强的单一特征。

3个反欺诈子分：
- FPD31子分：首期逾期31天专项预测
- 资金饥渴子分：多平台同时大量借款行为识别
- 失联子分：预测催收时用户是否失联

### 蒸馏系列（知识迁移）

蒸馏系列通过知识蒸馏将历史模型（V10/V11/V12）的预测能力迁移到V13子分中，V12蒸馏mob1 AUC=0.639，在历史模型中保持了较好的区分能力。

### 历史子分（74个，42个未入模）

历史子分按时间窗口切分（24h2/2425/25h1），共74个，其中42个因增益不足未入模，后续版本可进一步缩减，降低计算开销。
