# 双塔风险模型

> [!info]
> 双塔模型融合风险预测和业务意愿两个维度，实现"高意愿+低风险"用户优先放款，解决下单率下降背景下的精准筛选问题。

## STAR架构介绍

双塔模型采用STAR（Sampling Transformer for Recommendation）架构，核心公式：

$$\text{最终分} = (\text{风险塔分} + 0.03) \times \text{业务塔分}$$

> [!important]
> STAR架构的设计逻辑：风险塔预测"坏客户概率"，业务塔预测"下单/提现意愿"。乘积形式确保两个维度同时好才能得高分，缺一不可。偏置项+0.03防止风险塔分为0时乘积崩溃。

## 三种双塔模型

| 模型 | 业务塔标签 | 风险塔标签 | 应用场景 |
|------|-----------|-----------|---------|
| 下单风险双塔 | 7日内是否下单 | MOB1 7+ | 一次风控主要决策 |
| 提现风险双塔 | 是否提现 | MOB1 7+ | 通过审批后是否实际提现 |
| 5min提现风险双塔 | 5分钟内是否提现 | MOB1 7+ | 快速提现场景专用 |

## 下单风险双塔模型详情

| 项目 | 详情 |
|------|------|
| 架构 | STAR `(风险塔+0.03) × 下单塔` |
| 训练范围 | 20240101~20251131（较V12拉新5个月） |
| 特征数 | 1851个 |
| 下单AUC提升 | +1.5%（相比V12） |
| 风险AUC提升 | +0.2%（MOB1，相比V12） |

### 关键效果指标

| 指标 | 表现 |
|------|------|
| 头部bin金额逾期率 | 更低（高分段风险确实更低） |
| 头部bin额度使用率 | 更高（高分段用户意愿更强） |
| 相比V12下单AUC | +1.5% |
| 相比V12风险AUC | +0.2% |

> [!important]
> 双塔模型在下单率下降（76%→66%）的背景下价值凸显：传统风控只过滤坏客户，但好客户中也有大量不下单的，双塔模型进一步筛选出"高意愿"的好客户，提高资金利用率。

## 5min提现风险双塔模型

针对快速提现场景（5分钟内提现）的专用双塔模型：
- 特征数：300个
- 场景：用户通过审批后在极短时间内发起提现，行为模式异常，需专项建模
- 训练标签：5分钟内是否提现 × MOB1逾期
