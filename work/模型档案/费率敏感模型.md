# 费率敏感模型

> [!info]
> 费率敏感模型使用因果推断方法，识别对利率/期限组合敏感的用户，支持差异化定价策略，在合规利率约束下最大化业务量。

## 业务背景

| 项目 | 详情 |
|------|------|
| 合规约束 | 9期以上产品需做千2（0.2%日利率）及以下利率 |
| 目标客群 | CD评级（中等风险，有精细化运营空间） |
| 核心问题 | 哪些用户对降低利率/调整期限更敏感，愿意为优惠利率提高下单率？ |
| 业务价值 | 差异化定价，对敏感用户提供优惠利率提升下单，对不敏感用户保持正常利率 |

## Xlearner因果推断方法

> [!important]
> 费率敏感模型使用Xlearner（X-Learner）因果推断方法，属于Uplift Modeling的一种。核心目标是估计个体处理效应（ITE），而不仅仅是预测下单率。

### 方法论对比

| 方法 | 普通分类模型 | Xlearner因果推断 |
|------|------------|----------------|
| 预测目标 | 用户是否下单（P(下单)） | 降费对用户下单的增量影响（P(下单|降费) - P(下单|不降费)） |
| 反事实 | 不考虑 | 核心考虑（同一用户降费和不降费的反事实对比） |
| 适用场景 | 全量评分 | AB实验后的因果效应估计 |
| 输出 | 下单概率 | 个体处理效应（ITE），即uplift值 |

### Xlearner算法步骤

1. 用控制组（未降费）训练模型$\mu_0$，预测控制组下单率
2. 用实验组（降费）训练模型$\mu_1$，预测实验组下单率
3. 用$\mu_0$预测实验组的反事实结果，计算个体处理效应：$D_1 = Y_1 - \hat{\mu}_0(X_1)$
4. 用$\mu_1$预测控制组的反事实结果，计算：$D_0 = \hat{\mu}_1(X_0) - Y_0$
5. 最终ITE = 两组处理效应的加权融合

## 业务效果

| 指标 | 结果 |
|------|------|
| 意愿分bin后敏感人群下单率lift | 约1.5倍 |
| 核心筛选逻辑 | 意愿双塔划分低/中/高意愿 + 敏感模型精筛 |

> [!important]
> 费率敏感模型不是单独使用，而是与意愿双塔模型配合：先用双塔划分高/中/低意愿用户，再用敏感模型在各意愿层内识别对费率变化响应明显的用户，实现精准差异化定价。

## 期限利率产品配置

| 期数 | 利率要求 | 费率敏感模型策略 |
|------|---------|---------------|
| 1~6期 | 无强制上限 | 常规定价 |
| 9期 | 千2及以下（合规要求） | 在CD评级中识别高意愿敏感用户 |
| 12期 | 千2及以下（合规要求） | 同上，12期产品占比已增长至6.3% |
