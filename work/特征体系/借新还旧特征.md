# 借新还旧特征

## 核心目的

检测跨平台"拆东墙补西墙"行为——用户在A平台即将到期/已逾期时向B平台借新款偿还。这种行为是多头借贷的高风险信号，也是信贷违约的重要前兆。

---

## 四维度组合设计

借新还旧特征通过四个维度的笛卡尔积组合生成，约~800个变量：

| 维度 | 可选值 | 说明 |
|------|-------|------|
| 对比偏移(shift) | 前第1/2/3/4笔订单 | 按CREDIT_CreditStartDate排序，与当前订单对比 |
| 时间差窗口 | 前14/60/120天内 | 新订单开始日期在旧订单到期日前/后N天 |
| 筛选条件 | prev_closed / prev_overdue / amount_increased / creditor_changed | 上一笔是否结清？是否逾期？金额是否增？是否换新机构？ |
| 时间维度 | 6/12/24/all月 | 历史窗口 |

### 维度详解

**对比偏移(shift)**：
- shift=1：当前订单与前1笔订单比较
- shift=2：当前订单与前2笔订单比较
- shift=3/4：更早的历史订单
- 按 `CREDIT_CreditStartDate` 排序确定顺序

**时间差窗口**：
- before_14d：新订单在旧订单到期日前14天内开始（提前借新还旧）
- after_14d：新订单在旧订单到期日后14天内开始（逾期后紧急借款）
- 60d/120d：更宽的时间窗口，捕捉较长间隔的借新还旧

**筛选条件**：
| 条件 | 含义 | 风险信号 |
|------|------|---------|
| prev_closed | 前一笔已结清 | 结清后立即借新 = 以贷养贷 |
| prev_overdue | 前一笔有逾期 | 逾期后借新 = 资金链紧张 |
| amount_increased | 新借金额更大 | 借款金额膨胀 = 债务雪球 |
| creditor_changed | 换了新机构 | 换平台借 = 原平台可能已拒 |

## 命名模式

### 基础借新还旧标记

```
rollover_before_{window}_{shift}
rollover_after_{window}_{shift}
```

示例：
- `rollover_before_14d_1` -- 在前1笔订单到期前14天内开始了新订单
- `rollover_after_60d_2` -- 在前2笔订单到期后60天内开始了新订单

### 条件组合标记

```
{base}_prev_closed
{base}_prev_overdue
{base}_amount_increased
{base}_creditor_changed
```

示例：
- `rollover_before_14d_1_prev_closed` -- 前1笔已结清，且在到期前14天借新
- `rollover_after_60d_2_prev_overdue_creditor_changed` -- 前2笔逾期，换了新机构在到期后60天内借新

### 订单间时间差统计

```
days_between_orders_{type}_{shift}_{stat}_{window}
```

| type | 含义 |
|------|------|
| ss | 开始日期 - 开始日期（两笔订单的起始间隔） |
| sd | 开始日期 - 到期日期（新订单相对旧订单到期的提前/滞后） |
| dd | 到期日期 - 到期日期（两笔到期日的间隔） |

| stat | 含义 |
|------|------|
| mean | 平均天数差 |
| min | 最小天数差 |
| max | 最大天数差 |
| std | 标准差 |
| count | 符合条件的订单对数 |

示例：
- `days_between_orders_sd_1_mean_12m` -- 近12个月，新订单开始到前1笔到期的平均天数差
- `days_between_orders_ss_2_min_all` -- 历史全量，与前2笔订单开始日期的最小间隔

### 条件筛选后的时间差统计

```
{condition}_conditional_days_{type}_{shift}_{stat}_{window}
```

示例：
- `prev_closed_conditional_days_sd_1_mean_12m` -- 前1笔已结清的条件下，近12个月新订单开始到旧订单到期的平均天数差

## 高IV特征集

高IV特征集中在**条件时间差类**（IV~0.015-0.020），尤其是：

| 特征类型 | IV范围 | 说明 |
|---------|--------|------|
| 前1-4笔已结清订单的开始/到期日期差统计 | 0.015-0.020 | 最强信号 |
| 前1笔逾期+换机构的借新还旧标记 | 0.010-0.015 | 直接风险标记 |
| 订单间隔时间的标准差 | 0.008-0.012 | 借款节奏不稳定 |

**业务洞察**：
- 前1笔已结清的时间差特征IV最高，说明"结清即借新"是最强的以贷养贷信号
- shift=1（紧邻上一笔）的特征普遍比shift=3/4更有区分力
- 12个月窗口表现最好，6个月太短（样本不足），24个月太长（信号衰减）
