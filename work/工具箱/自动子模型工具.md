# autoSubModels - automatic submodels builder 
autoSubModels使用多进程，快速构建子模型，最高可开48进程。

**还在优化中，有任何错误欢迎指出**

# 教程1 -- 子模型构建工具 
- 此教程以极速贷子模型自动化为例，不使用征信变量

## 1 模型组合介绍
模型的构建有四个方向可以提供垂直性，分别是 base、target、变量、算法
* **base** 按照 评级、额度、征信、多头、收入和学历切分 
* **target** ever3q、ever8q、ever12q、ever12q_lk、回归target
* **变量** 分为四种：多头、自填、三方数据源1(旧)、三方数据源2（新增），四种变量可以有10种组合
* **算法** 常规、wce、金额加权、高多头坏人加权、主模型概率值加权wce

## 3 需要指定的的全局变量

- **dataframe**: ins,oos,oot,chuoe_df
- **每种变量类型包括的变量名**：vars_type1_feats,vars_type2_feats,...

- **变量类型到变量列表的映射**：

    ```python
    vartype_to_feats = {'vars_type1':vars_type1_feats,
                    'vars_type2':vars_type2_feats,
                    'vars_type1_vars_type2':vars_type1_feats+vars_type2_feats}
    ```

- **将评估指标名称映射到指标 rubik 字段**:

    eval_keys_dict = {'征信':'iszxgood','收入':'high_incom','多头':'alsnbankorgnum','学历':'high_edu'}
- **用于评估的target列表**：eval_target_list:list
- **分类变量列表**：categorical_feats:list
- **用于评估相关性的列名**：baseline_score:string
- **捞回和拒绝的阈值**：float：lh_threshold，jj_threshold(0.5, 3.5)
- **wce权重**：alpha：int
- **main_weighted 权重**：kpower, cap_val 
- **base_keys**：base组合列表 即 base_col_lst

## 4. 准备多进程函数参数列表              
                         
- **base组合列表**：base_col_lst，为1表示该样本用于训练
- **target列表**：train_target_list

- **变量组合列表**：vars_type_lst（即vartype_to_feats.keys()）
- **两个参数**：一个用于二分类 param、一个用于回归 param_reg
- **算法种类列表**：algorithm

- **argL**：每个子模型训练的参数、包含多个元组的列表 

    ```python
    argL1 = [(base_col, target, param, varstype, alg)
            for base_col in base_col_lst for target in train_target_list 
            for varstype in vars_type_lst for alg in algorithm if target not in ('aid19_ratio')]
    argL2 = [(base_col, target, param_reg, varstype, alg)
            for base_col in base_col_lst for target in train_target_list 
            for varstype in vars_type_lst for alg in algorithm if target in ('aid19_ratio')]
    #最终传入多进程函数的参数
    argL = argL1 + argL2
    ```

# 需注意的坑
**1.参数设置**

- num_threads 必须为1，否则多进程会锁死

**2.内存** 
- 注意根据占用内存大小来选择合适的核数，如果超出最大内存，也会导致多进程锁死
- 可以通过上文的方法，降低dataframe占用内存的大小
